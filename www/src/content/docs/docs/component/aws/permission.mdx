<Section type="about">
The AWS Permission Linkable helper is used to define the AWS permissions included with the
[`sst.Linkable`](/docs/component/linkable/) component.

```ts
sst.aws.permission({
  actions: ["lambda:InvokeFunction"],
  resources: ["*"]
})
```
</Section>
